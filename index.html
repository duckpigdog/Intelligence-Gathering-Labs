IG-LABS

红蓝队均可使用的信息收集靶场

Basic

Level 01

Whois 查询

Whois 查询是用于查询域名注册信息、IP 地址归属以及网站所有者等公开信息的网络工具
Basic

Level 02

Google Hack

Google Hack 是一种利用 Google 等搜索引擎的高级搜索语法和特性，来发现网络上未被公开或意外暴露的敏感信息、文件或系统漏洞的技术
Basic

Level 03

域名反查 IP

通过从全球不同地理位置的网络节点对同一个域名执行 Ping 或路由追踪，分析其返回的 IP 地址是否一致，来判断是否存在 CDN 或负载均衡
Basic

Level 04

端口扫描

端口扫描是一种向目标主机的网络端口发送探测数据包，以发现其开放端口及所运行服务的安全侦察技术
Medium

Level 05

指纹收集

指纹收集是通过分析目标反馈的独特特征信息，来精准识别其类型、版本、配置乃至具体漏洞的侦察技术
Medium

Level 06

子公司收集

子公司收集是通过商业数据库、公开情报、关联分析等手段，系统性地发现与目标母公司存在控股、投资或关联关系的所有下属公司或实体
Medium

Level 07

主域名收集

通过查询目标组织的官方 ICP/公安备案信息，直接、准确地获取其在国内合法运营的所有核心网站域名
Medium

Level 08

小程序收集

小程序收集（专指微信）是通过抓取微信公开搜索，系统发现目标机构开发运营的微信小程序资产
Medium

Level 09

公众号收集

微信公众号收集是通过微信搜索，系统发现并监控目标组织认证或运营的公众号及其关联资产
Medium

Level 10

服务号收集

服务号收集是通过微信搜索，系统发现目标组织认证的微信服务号
Medium

Level 11

APP 收集

APP 收集是通过官方应用商店、第三方分发平台，系统发现目标组织开发或运营的移动应用程序资产
Medium

Level 12

邮箱收集

邮箱收集是通过网络爬虫、公开数据泄露库、企业官网抓取、员工社交平台信息聚合等方式，批量获取与目标组织关联的电子邮件地址的侦察技术
Medium

Level 13

子域名收集

子域名收集是通过 DNS 爆破、证书透明日志、搜索引擎爬取等技术，系统性地发现目标主域名下所有二级、三级等子域名的侦察过程
High

Level 14

目录爆破

目录爆破是通过自动化工具，使用预设的字典对目标网站进行大量路径请求，以发现隐藏页面、后台入口、敏感文件或未授权访问接口的侦察技术
High

Level 15

JavaScript 接口泄露

JS 接口泄露是指 Web 应用前端代码中遗留的 API 密钥、内部接口地址、管理员路径等敏感信息，因未脱敏处理而被攻击者直接获取的常见安全漏洞
High

Level 16

GitHub 源码泄露

GitHub 泄露是指开发者误将密钥、密码等敏感信息上传到公开代码库，导致核心数据暴露的高危漏洞
High

Level 17

GitHub 分支泄露

GitHub 分支泄露是指开发者在未受保护的分支中提交了敏感代码或数据，导致其可通过直接访问分支或历史提交记录被公开获取的安全漏洞
High

Level 18

GitHub 提交泄露

GitHub 提交泄露是指：代码提交时误将敏感信息（如密码、密钥）一同推送至公开仓库，造成安全隐患
High

Level 19

GitHub 邮箱关联资产收集

通过邮箱在 GitHub 上搜索关联的代码提交、仓库和用户信息，来发现潜在泄露的敏感信息和资产痕迹
High

Level 20

GitHub API 使用

通过 GitHub API 列出组织下所有仓库的名称、URL、创建时间、最后更新时间等信息
High

Level 21

GitHub issues 泄露

GitHub issues 可能包含敏感信息、漏洞细节或内部讨论，若公开或配置不当会导致信息泄露风险
High

Level 22

.git 泄露

开发者将 .git 目录（包含版本控制信息、历史提交记录、配置文件等）上传至公开代码仓库，导致敏感信息泄露风险
High

Level 23

commit 提交溯源

commit 提交溯源可以通过分析提交历史、作者信息、时间戳和代码变更，追踪特定代码的来源、修改者的邮箱
High

Level 24

邮箱溯源

邮箱溯源可以通过关联公开数据、代码提交记录和社交平台信息，追踪特定邮箱背后的用户身份和活动轨迹
High

Level 25

联系方式信息收集

通过公开渠道（官网、电商、社交媒体等）获取目标机构/产品的联系方式，用于渗透测试、社会工程或客户验证
High

Level 26

ICP 备案查询

通过官方或第三方平台查询网站的备案信息，以确认其主办单位、许可证号及真实性，常用于资质核查或资产归属确认
High

Level 27

网络空间搜索引擎

通过主动扫描全网设备并索引其服务、端口及指纹信息，实现快速检索特定主机、服务或漏洞的专业搜索引擎
High

Level 28

SSL 证书搜集

通过证书透明日志、搜索引擎或扫描工具，大规模发现和收集目标域名、子域名及其相关主机的SSL证书信息，用于资产发现和安全分析
High

Level 29

君子协议

robots.txt 是一种文本文件，用于指示搜索引擎爬虫如何访问和索引网站的内容
